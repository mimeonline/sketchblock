# Contributing

Sketchblock welcomes focused bug reports, documentation improvements, and contributions that strengthen its Git-backed collaboration workflow.

Use discussions or issues for proposals and keep pull requests focused. Every code change must preserve architecture boundaries and pass the relevant typecheck, lint, test, and build gates.

By submitting a contribution, you agree that it is licensed under the Apache License 2.0 that covers this repository. The complete workflow lives in the repository's [CONTRIBUTING.md](https://github.com/mimeonline/sketchblock/blob/main/CONTRIBUTING.md).
